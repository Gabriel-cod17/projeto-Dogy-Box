Dogy Box — Website Institucional

Website institucional desenvolvido para o Dogy Box, com foco em presença digital, posicionamento de marca e conversão de leads.
O projeto apresenta os principais serviços do box, horários de treinos e canais de contato, oferecendo uma experiência moderna, responsiva e orientada ao usuário.

Visão Geral

O site do Dogy Box foi projetado para comunicar os valores da marca, destacar a proposta do box e facilitar o contato com novos alunos.
A solução foi desenvolvida com tecnologias web modernas, priorizando performance, usabilidade, responsividade e boas práticas de SEO.

Tecnologias

HTML5 — Estrutura semântica e otimizada para mecanismos de busca

CSS3 — Layout responsivo, Flexbox, Grid, animações e transições

JavaScript (Vanilla) — Interações, animações ao scroll e manipulação do DOM
